Trabalhando com listas

Listas ordenadas

Para criar listas ordenadas (Listas que a ordem é importante) nós utilizamos a tag OL e cada item da lista utilizamos a tag LI Para desgnar aquele item com um item da lista.

Podemos também configurar a lista para indicar qual caractere vai representar cada item e por qual carctere a lista irá iniciar. No exemplo abaixo o carcter é numerais e foi configurado para começar pelo número 3.

  1. Acordar
  2. Ir trabalhar
  3. Tomar café
  4. Ler o jornal
  5. Voltar ao trabalho
  6. Malhar
  7. Almoçar
  8. Tomar banho
  9. Estudar

Listas não ordenadas

Listas não ordenadas são listas que não necessitam de seguir uma ordem, como lista de compra a ordem não altera o resultado.

Para usar essa lista deve usar a tag UL
Podendo também configurar o carctere que representa o item.

  • Tomate
  • Limão
  • Carne
  • Pão

Lista de definição

As listas de definição são usadas para definir um termo, para isso se usa a tag DL. e dentro dessa tag se usa as tags DT para expor o termo que será definido. e DD para dar a definição como mostra o exemplo abaixo:

HTML
Linguagem de marcação para criação de conteúdo de um site.

Linguagem de marcação para o design de um site.
JavaScript
Linguagem de marcação para a interatividade de um site.

Mais configurações de listas

Colunas

Para dividir uma lista em colunas usamos o seletor columns e depois definimos a quantidade de colunas.

Caso os marcadores da lista estejam do lado de fora de um iside usa o seletor list-style-positionpara posicionalo dentro.

Estilo de marcadores de colunas

Para mudar os marcadores de uma coluna para algum diferente dos mensionados anteriormente use o seletor list-style-type: ''; e dentro das '\' cole o código do emeji a ser usado

Links Externos

Podemos criar links externos para outras páginas da web. Para isso, usamos a tag A, e dentro dessa tag você deve colocar o url da página.

Lembre-se de sempre usar as tag TARGET e REL para links externos.
